Seja muito bem-vindo!

Muito prazer, meu nome é Cláudio, sou cientista de dados, desenvolvedor e técnico em processamento de dados e aqui você poderá conhecer um pouco sobre o meu trabalho.


Utilizo Python, Django, Power BI, SQL, HTML e CSS para criar aplicações online personalizadas, extremamente seguras, velozes e escaláveis.

O que ofereço

A melhor maneira de administrar o seu negócio é ter informações detalhadas sobre ele. Registrar informações, sejam elas em cadernos, documentos, planilhas ou aplicações personalizadas é fundamental para a construção de indicadores.

Essas informações irão expandir a visão do seu negócio, contribuindo diretamente para o planejamento e tomada de decisões.

O meu trabalho vai desde a criação de ferramentas para a coleta dos dados até a criação de sites e dashboards personalizados.

Passo a passo

Tudo começa pela coleta dos dados, dados esses que hoje em dia deixaram de ser um diferencial entre os negócios e se tornou crucial para a sobrevivência no mercado. Com esses dados, inicia-se a fase exploratória, onde os dados são compreendidos e contextualizados para posteriormente serem padronizados para a criação de um modelo. Com esse modelo são criados os gráficos e dashboards que permitem uma visualização clara do seu negócio.

Coleta dos dados

A coleta pode ser feita por inúmeras ferramentas, sejam elas planilhas de Excel, arquivos CSV, JSON, etc. Dificilmente hoje em dia os dados são armazenados em papel, no entanto, se esse for o caso, apenas levará um pouco mais de tempo, mas não inviabilizará o processo.

Tratamento dos dados

Essa é a fase mais demorada em todo o processo de ciência dos dados. A limpeza e o tratamento é primordial para eliminar duplicidades, inconsistências, inconformidades, ausência de padrões ou até mesmo ausência de dados.

Criação dos relatórios

Aqui os dados ganham vida. Deixam de ser apenas dados e se tornam informações. Essas informações podem vir em gráficos de barras, de densidade, de dispersão, de setores ou em dashboards com vários desses elementos. Nessa fase surgem as evidências e as conclusões.

Python, Django, SQL, Power BI, HTML e CSS

Python é uma linguagem de programação de alto nível e de código aberto. Django é um framework para desenvolvimento rápido para web, escrito em Python. SQL é uma linguagem de consulta estruturada que serve para armazenar e processar informações em um banco de dados relacional. O Microsoft Power BI é um serviço de análise de negócios e análise de dados, que possibilita a criação de dashboards dinâmicos e interativos para a visualização dos seus dados. HTML é uma linguagem utilizada na construção de páginas na Web e o CSS é um mecanismo para adicionar estilos a uma página web.

Projetos

Aqui cito três projetos que foram meus objetos de estudo, sendo que um deles, o Livro de Parto, saiu do universo de estudo e hoje está rodando em ambiente de produção, colaborando para aprimorar o registro e gestão de indicadores de um serviço hospitalar. Além desses projetos, ao longo dos anos, criei centenas de relatórios e dashboards utilizando o Microsoft Power BI.

Agenda

Uma agenda on-line feita em Python com Django, com sistema de login, onde cada usuário visualiza apenas as suas informações. É possível adicionar novos usuários e cada usuário tem seu perfil com fotos.

Livro de Parto

Uma aplicação em Django para o registro de nascimentos em uma maternidade, acessada apenas por usuários registrados, onde cada um tem seu nível de acesso. As informações são inseridas pelos usuários através de um formulário moderno e intuitivo.

Blog

Um blog com layout moderno, voltado para leitura, onde é muito fácil adicionar novas postagens. Cada post pode ser configurado de forma independente, com sistema de busca através dos títulos, tags e palavras-chave.

Quanto custa?

Cada etapa na construção de um relatório ou de uma aplicação demanda tempo e esforços específicos. Tudo depende das informações que o(a) cliente fornece e do que ele(a) deseja como produto final. Além dos custos com o programador/cientista de dados, haverão os custos com servidor e hospedagem, no caso de uma aplicação on-line. Existem diversos provedores para esse serviço, dependendo da necessidade, com os mais variados preços. Acredite, sai bem mais caro para o seu negócio você não ter essas ferramentas!
